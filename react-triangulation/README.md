React Triangulation Demo

Quick start (requires Node.js 18+ / npm):

1. Install dependencies

```bash
cd react-triangulation
npm install
```

2. Run dev server

```bash
npm run dev
```

3. Open the URL printed by Vite (usually http://localhost:5173)

Notes:
- This is a small Vite + React demo containing the `EvaluationSystem` component.
- Data is stored in memory. For persistence integrate localStorage or a backend.
- Icons are from `lucide-react`.
